# Dental Software Build — Concept Note

> **Status: early concept archive. No application was implemented in this
> repository.**

This repository records an early idea: simplifying repetitive calculations
and record-keeping tasks for dental technicians through small, focused
software tools.

The first concrete result of that exploration became the
[Dental Alloy Weight Calculator](https://github.com/JasonLee0416/dental-alloy-calculator),
which converts wax-pattern weight and material-specific gravity into an
estimated casting alloy requirement.

## Why Keep This Repository?

It documents the starting point of a broader development approach:

1. observe a repetitive field workflow;
2. isolate the smallest costly or error-prone step;
3. build a narrow prototype;
4. validate the assumptions before expanding the product.

That same domain-first approach later informed projects for delivery
operations, including
[Routelo v2](https://github.com/JasonLee0416/Routelo.version_2).

For working code, visit the linked calculator repository.
